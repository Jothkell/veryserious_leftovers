splitting each project into its own Git repository. See `backup status` for a timestamped log of
every folder and the remote/bare repo it was pushed to.

(`datingapp1`, `dodger`, `fuku`, `imessage_tool`, `rosetta_decoy_converter`, etc.). When you are
ready to delete the folder, confirm each entry in `backup status` has an offsite backup and remove
these directories.
